🌹 BUY-COMP PORTAL

Set up & run live buy-competition leaderboards in a community's Telegram. The board posts hourly + answers /buyleaders; standings are provisional — official winners come from the Rose scan after the hold period.

🔑 Operator password

Your dedicated BUYCOMP_KEY (a scoped password just for this portal — not the master admin key). Stored locally in this browser only.
📖 How to run a competition
BEFORE YOU START
  • A group admin adds the Cluck Norris bot to the competition's Telegram group and makes it an admin — just "Post messages" + "Delete messages" rights (nothing else). This guarantees it can post the board and self-clean the old ones; a non-admin bot silently fails to post in restricted groups.
  • In that group, type /chatid — the bot replies with the group's numeric ID. Copy it.
  • Enter your operator password above.
1 · CREATE IT
Fill the New competition form:
  • Token mint + ticker — the token being bought.
  • Chat ID — paste from /chatid.
  • Metric — Cumulative (most bought in total) or Biggest single buy.
  • Start / End — your local time (leave start blank/now to begin immediately).
  • Hold period — hours winners must keep their buys after the end (0 = no hold).
  • Board refresh — how often the live board reposts (e.g. 60 min).
  • Prize places — one row per place, the amount each wins (add/remove rows).
  • Prize token — what winners get paid in: the comp token, USDC, SOL, or another SPL token.
Hit Create & Start → the live board posts to the group immediately.
2 · WHILE IT'S LIVE
  • The board self-reposts on your refresh interval (a fresh "comp is live" nudge) with a days/hours/minutes countdown.
  • Anyone in the group can type /buyleaders for instant live standings.
  • ↻ refresh board — force an immediate update · ■ end early — close before the end time · 🛑 STOP & alert — emergency cancel (alerts the group, no winners).
3 · AFTER IT ENDS → VERIFY
Once the hold period is over, hit ✅ verify winners. For each top wallet it checks on-chain:
  • ✅ held — qualified
  • 🛑 sold — disqualified (the next wallet promotes up)
  • ⚠️ transferred out — click trace to find the wallet that still holds the tokens (the "runner wallet") and use that one for payout
The verified list fills your prize places automatically.
4 · PAY OUT
  • 💸 open payout in airdropper — loads winners + amounts into the airdropper (in your prize token); review every recipient, then send from your wallet.
  • 🔗 client payout link — copies a link the project opens to pay winners from their own wallet (they keep custody — no tokens sent to you).
Payout pays the verified list only — never the raw live board (which is provisional and can't filter wash-trading in real time). Always trace ⚠️ transferred winners before paying. USDC / native / other-SPL pay cleanly; double-check the SOL path the first time.
